# Chapter 4 – Digifest 2025 ☑️ (+ Futuristic Study Planner 💡 Bonus)

## ✅ Mandatory: Digifest 2025

**Title:** *Digifest 2025 Landing Page*

A promotional page for a fictional digital festival, showcasing a clean layout and proper HTML5 structure.

Includes:
- A centered heading and intro paragraph inside a `.header` section
- An `h1` with the event name: **Digifest 2025**
- A `p` describing what the event is about
- A `section` with `h2` subheading titled “Line Up”
- An unordered list of  activities
- Proper semantic structure: `<header>`, `<section>`, `<ul>`, `<li>`
- Page styled with:
  - Centered text and content alignment
  - `text-align: center`, `max-width`, and `margin: auto`
  - Background color for the header
  - Font sizing and padding for readability
- Linked external CSS stylesheet: `style.css`
