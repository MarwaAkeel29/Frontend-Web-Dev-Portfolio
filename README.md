# 🌐 Frontend Web Development Portfolio

## 📖 About This Repository
This repository contains a comprehensive collection of frontend web development projects, structured across six progressive chapters. The projects document my journey from foundational HTML5 and CSS3 layouts to advanced CSS styling, responsive design, embedded media, and dynamic, logic-driven applications using vanilla JavaScript.

---

## 🛠️ Core Technologies
* **HTML5** (Semantic structuring, forms, embedded media)
* **CSS3** (Flexbox, Grid, animations, pseudo-classes, advanced shadowing)
* **JavaScript (ES6+)** (DOM manipulation, event listeners, game logic)

---

## 📂 Project Showcase

### 🍽️ Chapter 1: Foundations & Semantics
**Projects:** Recipe Page | HTML/CSS Quick-Reference Guide
* Built a fully styled recipe webpage demonstrating foundational HTML structuring (headings, lists, images) and basic CSS aesthetics.
* **Bonus:** Designed a color-coded, mobile-friendly Quick-Reference Guide card layout utilizing CSS for readability and semantic HTML for easy scanning.

### 🐉 Chapter 2: Theming & Multi-Page Architecture
**Projects:** How to Train Your Dragon Fan Site | Multi-Page Expansion
* Developed a cinematic, single-page fan website using core HTML/CSS concepts, Flexbox, and custom Google Fonts.
* **Bonus:** Expanded the site into a scalable multi-page experience with a shared CSS stylesheet, reusable top navigation bars, and visually themed hero sections.

### 🖼️ Chapter 3: Image Optimization & Advanced Styling
**Projects:** Timeless Treasures Image Gallery | "Styling Those Divs" Challenge
* Created a nostalgic image gallery demonstrating image handling (`alt`, `title`, aspect ratios), file size optimization, and CSS visual filters (sepia, contrast, brightness).
* **Bonus:** Showcased advanced CSS layering by recreating complex UI components using inner/outer `box-shadow`, `border-radius`, `outline`, and `text-shadow`.

### 🚀 Chapter 4: Layouts & UI Design
**Projects:** Digifest 2025 Landing Page | Futuristic Study Planner
* Structured a promotional landing page utilizing centered content alignment, max-widths, and clean semantic structures (`<header>`, `<section>`, `<ul>`).
* **Bonus:** Built a highly stylized, futuristic Study Planner featuring fixed sidebar navigation, CSS grid layouts, animated hover transitions, and neon `box-shadow` glowing effects.

### 📻 Chapter 5: Embedded Media Integration
**Projects:** Discover Turkey Embedded Video | Alan Partridge Soundboard
* Demonstrated responsive embedding of external YouTube iFrames alongside contextual text formatting.
* **Bonus:** Developed a themed, interactive audio soundboard using HTML5 `<audio>` tags and basic JavaScript-triggered playback within a static, non-scrolling UI grid.

### 🎮 Chapter 6: Interactive JavaScript Applications
**Projects:** Eco Drive Challenge | RGB Hacker Mode
* **Eco Drive:** Built a gamified petrol calculator that takes real-time user inputs to calculate fuel costs, emissions, and assigns dynamic "Eco Scores" and badges.
* **RGB Hacker:** Engineered a logic-driven color-matching game styled like a neon hacker terminal, featuring real-time DOM updates, dynamic state tracking (trace levels), and customized visual feedback (flicker/glitch animations).

2. Navigate to the specific Chapter folder you wish to view.
3. Open the corresponding `index.html` file in any modern web browser to view the project and interact with the applications.
