## 🌱 Mandatory Project: Eco Drive Challenge

**Eco Drive Challenge** is a gamified petrol calculator that helps users become more eco-conscious drivers by calculating fuel costs, emissions, and efficiency. It assigns badges, eco scores, and provides driving tips based on input.

### 🔧 Technologies Used

- **HTML** – for page structure and form layout.
- **CSS** – for a retro gaming aesthetic and responsive styling.
- **JavaScript** – for all dynamic calculations, badge logic, and tips.

### 🧠 Key Features

- Calculates **fuel cost**, **MPG**, and **CO₂ emissions**
- Assigns **Eco Score** based on results
- Rewards **badges** (e.g., Eco Hero, Fuel Waster)
- Displays **custom driving tips**
- Visually responsive and themed layout

### 🎨 Design Highlights

- Retro-gaming visuals with pixel art and colorful buttons
- Smooth animations and hover states
- Fun, motivational messages to encourage greener choices

## 💾 Bonus Project: RGB Hacker Mode

**RGB Hacker Mode** is a color-matching guessing game where players must visually identify the correct color based on an RGB code. Styled like a futuristic hacker terminal, the game uses trace levels and scoring to build tension.

### 🎮 How It Works

- A random RGB color code is generated
- Players pick one of three color boxes
- Correct guess increases access score
- Incorrect guess increases trace level
- Game ends when trace reaches 100%
- Restart button appears upon system breach

### 🔧 Technologies Used

- **HTML** – for interface structure
- **CSS** – for a neon retro-futuristic hacker aesthetic
- **JavaScript** – for game logic, random color generation, user feedback, and state tracking

### ✨ Game Features

- Real-time feedback via **messageOutput**
- Dynamic **score** and **trace level**
- Glowing neon **hover effects** on color boxes
- **Game Over screen** with restart option
- Animations like **flicker** and **glitch** for immersive UI
