# 🐉 HTML & CSS – Chapter 2 Project  
## How to Train Your Dragon Fan Site & Bonus Multi-Page Expansion

---

## 📄 Project Summary

This submission showcases the completed work for **Chapter 2** of the *Web Development* module. It comprises two components:

- **Core Project** – A fully styled **single-page fan website** themed around *How to Train Your Dragon*, built with foundational HTML and CSS.  
- **Bonus Feature** – An expanded, **multi-page version** with navigation and extra content using consistent theming.

---

## 🏰 Part One: HTTYD Fan Page (Core Project)

The goal of this task was to create a stylized, single-page website using only core HTML and CSS concepts.

### 🔧 Key Features Implemented:

- A clearly defined page title and custom favicon.  
- Semantic HTML structure with headings, paragraphs, and styled sections.    
- External **Google Fonts** chosen to suit the fantasy theme.  
- Consistent styling using CSS (color palette, fonts, layout, etc.).  
- Well-commented and readable HTML & CSS code.

### ✨ Enhancements and Creative Elements:  
- Carefully selected font combinations to match the cinematic atmosphere.  
- Button hover effects and color accents for interactivity.  
- Responsive layout using `flexbox` for a polished look.

> 💡 *This project helped build confidence in foundational layout and visual styling using only HTML and CSS.*

---

## 🌟 Part Two: Bonus Multi-Page Site

As an extension, the site was expanded into a **multi-page experience**, allowing users to explore themed sections like Overview Characters, Cast, Reviews, and more.

### 🧩 Highlights of the Bonus:

- Structured subpages inside a `pages/` directory for clear organization.
- A visually themed **hero section** featuring a background image and title.
- A bold **hero section** with overlay, styled title, and themed buttons.  
- A reusable **top navigation bar** to link across all pages.  
- Consistent theme across every page using a shared CSS stylesheet.  
- Additional content added to enrich the user experience.  
- Smooth navigation flow with hover styles and clear structure.

> 🧠 *This bonus section demonstrates the ability to structure and style a scalable, multi-page site while maintaining thematic and design consistency.*
