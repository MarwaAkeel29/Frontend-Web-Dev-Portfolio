# Chapter 5 – Embedding Content

## 🗺️ Mandatory Exercise: Embedded Video Page

### Project Title: *Discover Turkey – Cappadocia Adventure*

This project showcases how to embed video content into a webpage using HTML5. The page is themed around the stunning landscapes of **Cappadocia, Turkey**, and includes contextual content along with the embedded media.

### Key Features
- A centered embedded **YouTube video** (Cappadocia hot air balloon tour)
- Descriptive heading and paragraph text around the video
- Simple, responsive layout using semantic HTML
- Basic styling using external CSS for alignment and presentation

### Skills Practised
- Embedding external video using `<video>` (YouTube embed)
- Applying basic CSS for layout and spacing
- Aligning media and text for visual balance

## 📻 Bonus Exercise: Alan Partridge Soundboard

This task focused on embedding multiple **audio** clips into a fixed-layout soundboard, styled with a creative design. The bonus also allowed for extended customization.

---

### ✅ Basic Soundboard

#### Description
The basic version features a grid layout of audio samples from Alan Partridge. Each button plays a unique clip using HTML5's `<audio>` functionality, styled and structured to avoid page scrolling.

#### Features
- Static layout with no vertical scroll
- Multiple `<audio>` elements with `controls` or JS-triggered playback
- Styled audio buttons with clear labels
- Semantic HTML (`<main>`, `<section>`, `<button>`, etc.)

---

### 🌟 Extended Soundboard

#### Description
An upgraded version of the original soundboard with creative theming, interactivity, and visual polish.

#### Enhancements
- Themed color palette and typography
- Hover effects on buttons (e.g., glow, color shift)
- Categorized or more clearly grouped sound samples
- Visual styling to match a retro-radio or comic soundboard vibe
- Responsive layout
